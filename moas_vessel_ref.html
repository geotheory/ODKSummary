MOAS OPD Reference


Most data fields are fairly self explanatory. The Event number is a way to track which incident you are collecting data for - this should be the number of the incident that day - typically 1-3. To filter the data for the current incident you need to click on the bars to filter today's date and the current incident number.

If you need to download the data do so from ODK Aggregate. If you download a csv file that will open up in Excel. You'll notice the data fields don't contain the actual descriptions for e.g. nationality, but instead their respective codes. Fear not for it is very easy to lookup the actual values using this pre-prepared excel spreadsheet.
