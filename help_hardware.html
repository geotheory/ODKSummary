The Edison

The device is composed of the Intel Edison processor, three Sparkfun 'blocks' - the Base block (connecting to the Edison), Micro SD block and Battery block (next to the battery), a 2000 mAh battery (for est. 10-15 hours use), 3 micro-USB sockets (2 in the Base block and 1 in the Battery block), and a Micro SD card. It includes a WiFi router, for which the line-of-site range has been tested to 80 meters. We're running this as a tiny server to collect data in the field.

Basic operation

START UP the device by flicking the 'charging' switch on in the Battery Block to 'on'. Blue LEDs in the Base Block and Battery block will light up.

To TURN OFF the device, push the 'power' button (a very small black button) in the Base Block and keep pressed for 12 seconds. Now wait 60 seconds for the server to power down, and then flick the 'charging' switch off to disconnect the battery.

To CHARGE THE BATTERY, set the Battery Block switch to 'charging'/'off' and connect a USB power source (charger, power pack, phone, tablet) to the Battery Block socket. A red LED shows it's charging, and will disappear when fully charged. Charging won't power the system on. Please ensure any Edison battery is not left recharging too long - from empty a 6 hour charge should suffice - as overcharging can cause damage. Avoid exposing the Edison (battery) to temperatures over 40Â°C, especially with the box closed.

The SD CARD backs up data collected in csv format in case of a system failure. An SD card (formatted as ext4) should be inserted when the system is disconnected from power and it should mount automatically on boot-up. Data is automatically backed up to SD card.

TO CONNECT as a user look for a WiFi network like MissingMaps1 and connect with the password provided seperately. Your device is speaking to the server.

A system administrator can ACCESS THE SERVER either over wifi with ssh or via USB with the `screen` app. Instructions seperately.
